# CustomNickName
カスタムフォントを使用し、プレイヤーの名前を変更するデータパック


## 動作を確認しているバージョン
Minecraft Java Edition 1.21 ~


## 前提リソースパック
このデータパックは以下のリソースパックの導入を前提としています。

<b> [Negative Space Font](https://github.com/AmberWat/NegativeSpaceFont) </b>


## 使い方

  ### 基本
  1. apply
  2. clear
  
  ### あまり使わないもの
  3. cache
  4. init
